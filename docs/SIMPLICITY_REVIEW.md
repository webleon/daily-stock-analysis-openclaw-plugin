## 🎯 桥接插件职责分析

### 桥接插件的理想职责

**应该做：**
1. ✅ 检测 DSA 服务是否运行
2. ✅ 调用 DSA API 获取分析结果
3. ✅ 简单的部署命令（可选）
4. ✅ 关键词触发检测

2. ❌ 交互式配置向导
3. ❌ 依赖管理
4. ❌ 虚拟环境创建
5. ❌ API Key 获取指引
